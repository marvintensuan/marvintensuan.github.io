I also volunteer for causes related to saving the environment, as well as promoting a safe workplace environment for everyone.

When I was 12, I looked up on Yahoo! the words "how to make my own video game". The rest was history.

As an accountant, I help drive efficiencies in my processes largely thanks to Visual Basic for Applications, Stack Overflow's Most Dreaded Language for 2019 and 2020, and VBScript for SAP GUI scripting.

I've known Python for _ years now. Language aside, I love the community behind it and its massive resources available online.

At the moment, I am learning a compiled, statically-typed language Kotlin. Other languages I've been exposed to include Octave (as used in Andrew Ng's Machine Learning); Pascal (first language taught to me in formal instruction); HTML, CSS and Javascript (for creating websites); and last but not the least, Scratch for CS50's Week 0 Project.

The bulk of programming tasks I do as a professional is related to developing reusable scripts. I started to use Python soon as I discovered the difference in productivity it gave me.

Due to its prominence in the Data Science community, Python has since offered me tools like Jupyter Notebooks which redefined my work. This pushed me to take a look at other domains in which Python is good for, like web development and scientific computing.

Overall, I am enthralled by Python not just as a language but as a big community.
